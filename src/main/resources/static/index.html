Download & Extract Log Files

Select a predefined location or enter custom URLs. The system will auto-detect and download:
• File URLs: Downloads the specific file
• Directory URLs: Discovers and downloads all log files (.log, .gz, .zip, and extensionless text files)
• Smart filtering: Skips binary files and 0-byte files, detects text files automatically
Compressed files (.gz, .zip) are automatically extracted.
